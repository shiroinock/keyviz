# プロジェクト構造チェック

## 説明
不用意なディレクトリやファイルが追加されていないか確認します。

## 適用条件
- 新しいディレクトリが作成された
- 新しいファイルが追加された
- ファイルが移動・リネームされた

## チェック項目
- [ ] `src/` 以下のディレクトリが CLAUDE.md の仕様に従っているか
- [ ] ファイルが適切なディレクトリに配置されているか
- [ ] コンポーネントは `src/components/` 以下にあるか
- [ ] ユーティリティ関数は `src/utils/` にあるか
- [ ] データ定義は `src/data/` にあるか

## 重要度
high
