# テスト品質チェック

## 説明
テストファイルの品質を確認します。

## 適用条件
- `.test.ts` または `.test.tsx` で終わるファイル
- `src/__tests__/` ディレクトリ内のファイル

## チェック項目

### 1. テストと実装の整合性
- [ ] テスト名が実際の検証内容を正確に反映しているか
- [ ] モックが正しく機能しているか

### 2. テストの重複チェック
- [ ] 同じことを検証するテストが複数存在していないか

### 3. Arrange-Act-Assert パターン
- [ ] テストが AAA パターンに従っているか

## 重要度
high

## 判定基準
- **PASS**: すべての観点で問題なし
- **WARN**: テスト名と内容の不一致、テストの重複
- **FAIL**: テストが実質的に機能していない
