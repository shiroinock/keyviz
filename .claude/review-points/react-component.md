# React コンポーネントレビュー観点

## チェック項目

### 基本構造【必須】
- [ ] コンポーネントの責務が単一か
- [ ] 適切な命名規則（PascalCase）

### Props【重要】
- [ ] Props型が定義されているか
- [ ] 不要なオプショナルプロパティがないか

### フック使用【重要】
- [ ] 依存配列の完全性
- [ ] 不要な再レンダリングの回避

### アクセシビリティ【重要】
- [ ] セマンティックHTMLの使用
- [ ] 適切なARIA属性

## 判定基準
- **FAIL**: Props型未定義、依存配列不足
- **WARN**: 複数責務、keyにインデックス使用
- **PASS**: すべての必須項目をクリア
